# markitdown desktop app

A Tauri 2 GUI wrapper around the `markitdown` CLI. Drop a file, get clean
Markdown. The optional "Strip boilerplate" toggle runs the post-processor's
opt-in cleanup pass.

## Requirements

- Node.js 20+ and npm
- Rust toolchain (`rustup`, stable)
- The `markitdown` CLI on PATH:
  ```sh
  pip install markitdown-postproc
  ```
  The app launches whichever of `markitdown`, `markitdown.exe`,
  `py -m markitdown`, or `python -m markitdown` resolves first.

## Develop

```sh
cd app
npm install
npm run tauri dev
```

## Build installers

```sh
cd app
npm install
npm run tauri build
```

Output lands in `app/src-tauri/target/release/bundle/`:

- Windows: `.msi` (WiX), `.exe` (NSIS)
- macOS: `.dmg`, `.app`
- Linux: `.deb`, `.AppImage`

CI builds installers for all three platforms on every push to `main` —
see `.github/workflows/build-desktop.yml`.

## Icons

Placeholders generated by `app/scripts/gen_icons.py` (requires Pillow).
Regenerate with:

```sh
python app/scripts/gen_icons.py
```
