markitdown-postproc

Convert PDF, DOCX, XLSX, PPTX, HTML, audio, and more to clean Markdown — with an automatic offline post-processing pass that repairs broken syntax, normalizes tables, and strips boilerplate.

View on GitHub Download .exe pip install

Install

pip install markitdown-postproc

The distribution name is markitdown-postproc (the upstream markitdown name is owned by Microsoft). The import name is unchanged.

Use

from markitdown import MarkItDown

md = MarkItDown(strip_boilerplate=True)
result = md.convert("report.pdf")
print(result.text_content)

What the post-processor does

fix_broken_markdown

Closes unclosed **bold** / _italic_ spans (ignoring snake_case), adds missing spaces after heading hashes, closes broken links, collapses 3+ blank lines, and strips trailing whitespace.

fix_markdown_tables

Pads ragged rows, guarantees a separator row, drops duplicate headers, trims cell whitespace, and marks merged interior cells with [merged].

strip_boilerplate opt-in

Removes page numbers (1, Page 2, - 3 -), headers/footers repeated 3+ times, legal boilerplate ("All rights reserved", "Confidential"), and punctuation-only lines. Table separators are preserved.

Pipeline order

raw output
   -> fix_markdown_tables()
   -> fix_broken_markdown()
   -> strip_boilerplate()   # only if strip_boilerplate=True
   -> final markdown

Fully offline. No external API calls.
